# Тесты проекта

### Структура тестов

```
tests/
├── components/    # Тесты компонентов Vue
├── stores/        # Тесты Pinia stores
└── utils/         # Тесты утилит
```

### Запуск тестов

```bash
# Запуск всех тестов
npm test
```

### Конфигурация

Конфигурация Vitest находится в `vite.config.ts`:

- `globals: true` - глобальные функции тестирования доступны без импорта
- `environment: 'jsdom'` - окружение для тестирования DOM компонентов
- Алиас `@` настроен для импорта из `src/`
